Publications

Ecology-journal style list generated from data/publications.json. Edit that file to add or update entries; this page will update automatically.

How to add a new paper

  1. Open data/publications.json.
  2. Copy an existing item and update fields (authors, year, title, journal, status, doi/url/preprint).
  3. Refresh this page.

If you prefer, you can also delete the JSON/JS system and maintain a manual list—this template is intentionally simple.
